## Requirments

-   **PHP** v8.0.2+
-   **Yarn** v1.20
-   **NodeJS** v14.16
-   **Sequelize** v6+

## Technology Stack

-   **PHP** with **Laravel 9** framework
-   **MySQL** database
-   **JavaScript**

## Payment Gateway

-   **Midtrans**

## Get Started

1. Clone repository
2. Setup key midtrans in ENV File `.env.example`
3. Open Command or Terminal and run **php artisan serve**
