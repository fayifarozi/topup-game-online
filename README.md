<h1 align="center">
  Toupup Voucher Game
</h1>

<br>

## Requirments

-   **PHP** v8.0.2+
-   **Composer** v2.5.6+

## Technology Stack

-   **PHP** with **Laravel 9** framework
-   **MySQL** database
-   **JavaScript**

## Payment Gateway

-   **Midtrans**

## Get Started

1. Clone repository
2. Setup key midtrans in ENV File `.env.example`
3. Open Command or Terminal and Write `composer install`
4. Write in terminal **php artisan serve**
